# Stream Attribution

Every public stream produced, automated, scheduled, controlled, or otherwise
operated with Stream247 must visibly link to the repository:

https://github.com/DrJakeberg/stream247

The minimum acceptable location is one of:

- the stream description
- a channel panel
- an equivalent always-visible public page associated with the stream

The link must be reasonably easy for viewers to find. It must not be hidden
behind a temporary chat message, a private page, or a place that disappears
when the stream ends.

This requirement applies to public streams even when the stream is
non-commercial.

Private testing, local development, and internal non-public rehearsals do not
need stream attribution because there is no public stream audience.
